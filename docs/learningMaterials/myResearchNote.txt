    5.3 Importance of Shared Memory System
        How is shared memory different from orchestrator in their distinctive funcionalities?
        ╔══════════════════════╦═══════════════════════════╦═══════════════════════════╗
        ║                      ║    SHARED MEMORY          ║   ORCHESTRATOR AGENT      ║
        ╠══════════════════════╬═══════════════════════════╬═══════════════════════════╣
        ║ What is it?          ║ Database / Storage        ║ Strategic Decision-Maker  ║
        ║                      ║ (Like Google Drive)       ║ (Like Project Manager)    ║
        ╠══════════════════════╬═══════════════════════════╬═══════════════════════════╣
        ║ Has AI/LLM?          ║ No                        ║ Yes                       ║
        ╠══════════════════════╬═══════════════════════════╬═══════════════════════════╣
        ║ Makes Decisions?     ║ No (passive)              ║ Yes (active)              ║
        ╠══════════════════════╬═══════════════════════════╬═══════════════════════════╣
        ║ Main Function        ║ Store & Retrieve Data     ║ Coordinate & Decide       ║
        ╠══════════════════════╬═══════════════════════════╬═══════════════════════════╣
        ║ Example Methods      ║ • write(agent, key, val)  ║ • determine_strategy()    ║
        ║                      ║ • read(agent, key)        ║ • coordinate_agents()     ║
        ║                      ║ • store_episode()         ║ • resolve_conflict()      ║
        ║                      ║ • get_learned_pattern()   ║ • evaluate_results()      ║
        ╠══════════════════════╬═══════════════════════════╬═══════════════════════════╣
        ║ Uses the other?      ║ No                        ║ Yes (uses memory)         ║
        ╠══════════════════════╬═══════════════════════════╬═══════════════════════════╣
        ║ For the paper        ║ "Shared episodic memory   ║ "Intelligent meta-agent   ║
        ║                      ║  enables learning"        ║  for coordination"        ║
        ╠══════════════════════╬═══════════════════════════╬═══════════════════════════╣
        ║ Code File            ║ shared_memory.py          ║ orchestrator_agent.py     ║
        ╠══════════════════════╬═══════════════════════════╬═══════════════════════════╣
        ║ Analogy              ║ Library (stores books)    ║ Librarian (helps you)     ║
        ║                      ║ Whiteboard (displays)     ║ Teacher (uses board)      ║
        ║                      ║ Hard drive (storage)      ║ Operating system (manages)║
